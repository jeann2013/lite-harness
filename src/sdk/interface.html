lite-harness SDK — drop-in compatible with Claude Agent SDK

Identical public API. The only line that changes when you migrate is the import.

API shape

Python exposes a stateful ClaudeSDKClient class; JS/TS exposes only query() returning a controllable async generator. lite-harness mirrors each language's native shape exactly — so existing Claude Agent SDK code runs unchanged after swapping the import.

One-shot query()

async with ClaudeSDKClient(options=ClaudeAgentOptions()) as client:    await client.query("fix auth.py")    async for message in client.receive_response():        print(message)    await client.interrupt()
